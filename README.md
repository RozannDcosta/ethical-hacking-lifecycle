# Reconnaissance and Footprinting – Ethical Hacking (CEH)

This repository documents my hands-on learning and notes for **Reconnaissance and Footprinting**, the crucial first phase of ethical hacking and penetration testing.

The focus is on **information gathering**, understanding **attack surfaces**, and learning how attackers collect data before launching attacks. The goal is to understand these techniques to better defend against them.
