# Reconnaissance Lab – Passive Information Gathering

## Objective
To understand how reconnaissance is used to gather publicly available
information in an ethical hacking context.

## Background
Reconnaissance is the first phase of the ethical hacking lifecycle
and focuses on collecting information before any attack occurs.

## Tools Used
- Google Search Operators
- WHOIS
- DNS Lookup
- nslookup

## Lab Environment
- OS: Windows
- Target: example.com (public test domain)

## Methodology / Steps
1. Selected a public test domain
2. Performed domain registration lookup
3. Enumerated DNS records
4. Reviewed publicly available website data

## Observations
- Public records can reveal technical and organizational data
- DNS misconfigurations may expose infrastructure details

## Learning Outcome
- Gained hands-on experience in reconnaissance
- Understood the importance of OSINT
- Learned ethical lab documentation

## Security Relevance
Organizations must regularly monitor publicly available information
to reduce attack surface.

## Disclaimer
This lab was performed strictly for educational purposes.
