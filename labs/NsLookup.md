# DNS Enumeration Lab – nslookup

## Objective
To perform DNS enumeration using nslookup and analyze publicly available DNS records of a domain.

## Tools Used
- nslookup (Command-line DNS query tool)
  
## Lab Environment
- Operating System: Kali Linux
- Tool Used: Terminal
- Target Domain: example.com (public test domain)

## Background
DNS (Domain Name System) translates domain names into IP addresses.
DNS enumeration helps identify infrastructure components such as mail servers, name servers, and IP mappings.
